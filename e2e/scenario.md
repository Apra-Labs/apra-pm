# pm e2e -- sprint on the toy

Use the **pm** skill to run one sprint. Inputs the skill needs:

- **Repo:** already cloned at `{{REPO}}` (base `main`, remote `origin` -> the toy).
- **Branch:** `{{BRANCH}}`.
- **Requirement:** all open P1 issues from `bd ready` in `{{REPO}}` (take however many are shown).

Then run the pm commands in order: **plan** (with those issues as the
requirement), **start**, **cleanup**.
